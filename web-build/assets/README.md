Content Assets
==============

This folder contains all static files which should be used inside pages and articles.

> remember to avoid white spaces or special characters in assets files!

```
// good
image.jpg
my_image.jpg
my-image.jpg
my.image.jpg

// bad
"my image.jpg"

// very bad
"my:image.jpg"
"my~image.jpg"
"my%image.jpg"

```
